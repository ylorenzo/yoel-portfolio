Habilidades del proyecto

  • Front-End: Windows Forms
  • Back-End: Windows Forms
  • Bases de datos: Oracle, SQLite
  • Lenguajes de programación: Java, C#
  • Sistemas operativos: Windows, Android

Descripción del proyecto

CATASTRO es básicamente una aplicación de Android desarrollada para recopilar en el terreno la información de las parcelas y luego actualizar a través de una aplicación de escritorio la base de datos del SISCAT (el sistema informativo del catastro nacional de Cuba).
